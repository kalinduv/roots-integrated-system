Roots Integrated System

An Integrated Institute Management System developed as a group project for a university assignment. The system streamlines academic and administrative operations by providing modules for managing students, staff, courses, attendance, examinations, results, and fee payments.

Features

* Student Management
* Staff & Teacher Management
* Course Management
* Attendance Tracking
* Examination Management
* Results Management
* Fees & Payment Management
* Receipt Generation
* Email Notifications


Technologies Used

Frontend

* React.js
* Vite
* CSS

Backend

* Node.js
* Express.js

Database

* Firebase Firestore

Additional Services

* Nodemailer (Email Notifications)
