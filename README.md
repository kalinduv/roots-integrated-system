
# Roots Institute Integrated System

This project merges the six separate module projects into **one frontend** and **one backend**.

## Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: Firebase Firestore

## Run
### Backend
```bash
cd backend
npm install
npm start
```
Backend runs on **http://localhost:5001**.

### Frontend
Open a second terminal:
```bash
cd frontend
npm install
npm run dev
```
Open the Vite URL shown in the terminal, usually **http://localhost:5173**.

## Included modules
- Student Management
- Course Management
- Attendance Management
- Exams & Results Management
- Fee & Payments Management
- User Management & Dashboard

## Note
Port **5001** is used to avoid the macOS AirPlay conflict on port 5000.
